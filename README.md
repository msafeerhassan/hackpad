# My Cool Hackpad
## Project Details
My hackpad consist of 4 keys, a 0.91" OLED Display and two SK6812-MINI LED'S. Moreover, I am using XIAO RP2040. The case will be 3D printed while the the PCB will be printed and shipped by JLCPCB. I am going to hand solder all the componenets. PCB and Schematic Diagram is designed in KiCad while the Case is designed in Fusion360.
